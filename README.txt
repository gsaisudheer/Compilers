Scanner:
   compile the following way:
	 $ gcc scanner.c util.c
     This produces a ./a.out
     Run the executable as follows:
     $ ./a.out fullSample.tny
     The output will be produced in the output.tny file
     
Parser:
    compile as follows:
     $ gcc parser.c util.c
     This produces a ./a.out
     Run the exectuable as follows:
     $ ./a.out fullSample.tny
     If there are no errors, the program quits without displaying any error message
